# Client Query Management System

A web-based application for managing client queries efficiently.

##  Live Demo
 https://clientquery.streamlit.app

## Technologies Used
- Python
- Streamlit
- MySQL (Railway Cloud)
- Pandas
- Hashlib (SHA-256)

##  Features
- Secure Login & Registration
- Client Query Submission
- Support Team Dashboard
- Real-time Query Status Updates
- Password Hashing for Security

##  Database
- Hosted on Railway Cloud MySQL
- Tables: users, queries

##  How to Run Locally
1. Clone repo
2. Install requirements: pip install -r requirements.txt
3. Run: streamlit run Login.py
